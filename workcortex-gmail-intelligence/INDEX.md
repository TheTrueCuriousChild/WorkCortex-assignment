## ğŸ”§ How It Works

### Execution Flow
1. User clicks "Start" in Streamlit
2. Backend creates Engine and registers tasks
3. Engine runs each task sequentially
4. Each task emits STARTED â†’ (work) â†’ SUCCESS/FAILED
5. Events go to in-memory queue
6. UI polls queue and displays live table
7. Results show in UI (Excel preview, download)

### Event Format
```json
{
  "timestamp": "2025-12-25 01:13:42",
  "order": 1,
  "step": "Fetching Gmail Emails",
  "tool": "Gmail API",
  "status": "STARTED"
}
```
